# portal-init-hub

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面文件。这些页面可能包括静态展示页、工具页面、信息聚合页等，它们彼此之间没有强依赖关系，每个页面都可以单独访问或使用。

仓库不绑定任何特定域名或网站，仅作为页面资源的存储与分发中心。

## 目录说明

```
portal-init-hub/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
├── docs/           # 项目文档或说明文件
└── README.md       # 本文件
```

- **pages/**: 每个 HTML 文件独立存放，命名建议使用英文或拼音，避免特殊字符。
- **assets/**: 如果页面需要共享样式或脚本，可放置于此。若页面完全自包含，此目录可为空。
- **docs/**: 可存放与页面内容相关的补充说明、使用指南等。

## 页面归档说明

- 所有 HTML 页面应保持结构完整，可独立运行。
- 页面内部引用的资源建议使用相对路径，或直接内联。
- 提交新页面时，请在 pages/ 目录下新增文件，并确保文件名不重复。
- 建议在页面头部添加简要注释，说明页面用途或版本。

## 维护说明

- 本仓库欢迎贡献者提交新的 HTML 页面或改进现有页面。
- 提交前请检查页面在主流浏览器中的基本兼容性。
- 不建议在页面中包含外部跟踪脚本、广告或恶意代码。
- 如有重大结构变更，请先在 issues 中讨论。

## 使用方式

1. 克隆或下载本仓库。
2. 在 pages/ 目录中找到需要的 HTML 文件，直接用浏览器打开即可查看。
3. 若需部署，可将整个仓库上传至静态托管服务，pages/ 目录下的页面可直接通过 URL 访问。
